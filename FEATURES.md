# ✨ Request Buddy - Features

## 🎯 Core Features

### API Testing
- ✅ HTTP request builder (GET, POST, PUT, DELETE, PATCH, etc.)
- ✅ Request headers management
- ✅ Query parameters
- ✅ Request body (JSON, Form Data, Raw)
- ✅ Authentication (Bearer Token, Basic Auth, API Key)
- ✅ Response viewer with syntax highlighting
- ✅ Response time and status tracking

### Collections
- ✅ Organize requests into collections
- ✅ Create, edit, delete collections
- ✅ Import/Export Postman collections
- ✅ Nested folder structure
- ✅ Drag and drop organization

### Environments
- ✅ Multiple environment support
- ✅ Environment variables
- ✅ Variable resolution in requests
- ✅ Quick environment switching
- ✅ Global and local variables

### Request History
- ✅ Automatic request history tracking
- ✅ Search and filter history
- ✅ Replay previous requests
- ✅ Clear history
